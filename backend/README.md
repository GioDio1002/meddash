# MedDash Backend

FastAPI backend for the clinician-assistive MedDash prototype.

## Current implementation status

- The app currently persists patients and consult sessions in `backend.store.InMemoryStore`.
- `psycopg` and `redis` are already declared dependencies, and `compose.yaml` provisions PostgreSQL plus Redis, but the runtime migration is not finished yet.
- RAG citations and diagnosis outputs are generated by orchestrator logic and are not yet backed by durable data stores.

## Run

```bash
uv run --directory backend uvicorn backend.app:app --reload --host 0.0.0.0 --port 8000
```

## Test

```bash
uv run --directory backend pytest
uv run --directory backend ruff check .
```
